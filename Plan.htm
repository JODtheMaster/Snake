Planning document for Python snake game

Using module pygame , as well as my Utilities module from an earlier project that I made at home.

We will create several different .py files for each element of our game.
They will be the:

  • main
  • snake
  • variables
  • board
  • graphics

Contents of each file:

Main
This will contain the window creation/updating, exit loop handling, and calls all the functions from other modules. This should be quite a short file.
The only functions we will use here are Controller (a class for the entire main loop), the event loop (which consists of several different files)
Snake
This module contains the class that holds the data for each snake segment. What we shall do is to create a segment object in each division of the board.
